# Bilibili Quality Filter - Testing Guide

## 测试检查清单

### 1. 安装测试

- [ ] 从 `chrome://extensions/` 加载已解压的扩展
- [ ] 验证图标正确显示
- [ ] 验证扩展名称和描述显示正确

### 2. 弹出窗口测试 (Popup)

- [ ] 点击工具栏图标打开弹出窗口
- [ ] 测试主开关启用/禁用
- [ ] 测试各过滤选项开关
- [ ] 验证统计信息加载
- [ ] 点击 "Open Settings" 打开设置页

### 3. 设置页测试 (Options)

#### 常规设置
- [ ] 启用/禁用过滤器
- [ ] 切换"淡化而非隐藏"选项
- [ ] 切换"自动折叠评论"选项
- [ ] 切换"显示屏蔽用户按钮"选项

#### 过滤器设置
- [ ] 启用/禁用引战内容过滤
- [ ] 启用/禁用标题党过滤
- [ ] 启用/禁用同质化过滤
- [ ] 启用/禁用评论过滤

#### 关键词管理
- [ ] 添加自定义关键词到各分类
- [ ] 删除已有关键词
- [ ] 切换分类标签查看关键词

#### 黑名单管理
- [ ] 在B站评论上点击"屏蔽用户"
- [ ] 验证用户出现在黑名单中
- [ ] 取消屏蔽用户

#### 数据管理
- [ ] 导出数据为 JSON 文件
- [ ] 从 JSON 文件导入数据
- [ ] 清除所有数据

### 4. 内容过滤测试

#### 视频过滤
- [ ] 访问 B站首页，检查低质量视频是否被隐藏/淡化
- [ ] 访问搜索页面，检查过滤效果
- [ ] 验证被隐藏视频不占用布局空间
- [ ] 验证淡化视频悬停时恢复透明度

#### 评论过滤
- [ ] 打开视频页面，检查评论区
- [ ] 验证低质量评论被折叠
- [ ] 点击"仍然显示"展开评论
- [ ] 点击"屏蔽用户"屏蔽评论者

### 5. 跨标签页同步测试
- [ ] 在标签页A屏蔽用户
- [ ] 切换到标签页B，验证该用户内容被隐藏
- [ ] 在设置页修改设置，验证所有标签页生效

### 6. 性能测试
- [ ] 滚动页面时观察是否有卡顿
- [ ] 检查控制台是否有错误信息
- [ ] 长时间使用观察内存占用

### 7. 国际化测试
- [ ] 切换浏览器语言为英文，验证界面显示
- [ ] 切换浏览器语言为中文，验证界面显示

## 常见问题排查

### 扩展未生效
1. 检查 `manifest.json` 是否有效
2. 检查控制台是否有加载错误
3. 验证 `content_scripts` 匹配规则是否正确

### 过滤不准确
1. 检查 `storage/blocklist-manager.js` 是否正确加载默认关键词
2. 验证 IndexedDB 是否正常初始化
3. 检查评分算法参数

### UI 显示异常
1. 检查 CSS 是否被其他样式覆盖
2. 验证 CSS 类名是否正确应用
3. 检查 B站 DOM 结构变化

## 开发调试

### 开启调试日志
在控制台运行：
```javascript
localStorage.setItem('BQF_DEBUG', 'true');
```

### 检查 IndexedDB
```javascript
// 查看数据库
const db = await indexedDB.open('BilibiliQualityFilterDB');

// 查看关键词
const tx = db.transaction('keywords', 'readonly');
const store = tx.objectStore('keywords');
const keywords = await store.getAll();
console.log(keywords);
```

### 重置扩展
1. 清除 IndexedDB
2. 清除 chrome.storage.sync
3. 重新加载扩展
